# Trips  
_AGH University of Science and Technology, Cracow 2022/2023_

 Angular project for web application university course by **Micha≈Ç Godek**  
   

  

After git clone run 
 ~~~
 npm install
 ng serve
 ~~~

Application uses google's firebase modules:
 * firestore
 * authentication


Implemented functionalities:
* Navigation with angular routing
* user registration and login
* user roles
* trips
  * browsing
  * filtering
  * reserving, buying
  * rating, commenting
